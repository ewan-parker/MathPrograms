## Math Programs

A collection of **console‑based Java programs** for common mathematical operations.

Includes: 

**Matrix Multiplication:** Multiply two matrices after validating their dimensions. 
[See example output](MatriceMultiplication/exampleOutput.txt)

**Matrix Solver:** Perform Gauss‑Jordan elimination to reduce a matrix (or augmented system) to row‑reduced form.
[See example output](MatrixSolver/exampleOutput.txt)
